Your app is ready!

All files have been written to your project. Here's what to do next:

1
Open master-prompt.md in Cursor

The file is highlighted in your file explorer. Right-click it → Open With → Cursor.

2
Open a new Agent chat in Cursor

Press Cmd/Ctrl+L, switch to Agent mode, and paste (or drag in) the master-prompt.md content.

3
The AI completes automated setup

It configures your app, applies database migrations via Supabase MCP, scaffolds tabs, and writes SETUP_TODO.md with the remaining manual steps.

4
Complete the manual steps

Open SETUP_TODO.md — it lists only what the AI couldn't do (SMTP config, first admin user, deploy).
